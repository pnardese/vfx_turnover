Export AAF with VFX clip notes using vfx-turnover.

Takes an AAF file path as argument. Run `vfx-turnover -e $ARGUMENTS` to import the AAF, then run `vfx-turnover -a` interactively, prompting the user for: AVID user name, marker color, marker position, and clip color. Confirm overwrite if output file already exists. Output is saved as `<stem>_new.aaf` next to the source AAF.
